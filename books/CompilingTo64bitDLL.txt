Follow these instructions to build CodeBase libraries on Windows.

Files are provided in this repository to allow compilation by Microsoft Visual Studio Version 2017, 2019, and 2022.

The build directory in this repository contains a set of 4 subdirectories that provide c4dll.vcxproj files with
appropriate configurations to support modern versions of Visual Studio.  

For more details on how to build CodeBase applications with Visual Studio from original source, see the 
readme.md for this repository and the readme.txt in the WorkingSource subdirectory.
